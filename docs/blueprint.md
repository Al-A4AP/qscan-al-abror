# **App Name**: Al-Abror Q

## Core Features:

- QR Smart Validator: High-speed smartphone scanner tool for real-time coupon verification and automated check-ins.
- Live Operations Dashboard: Real-time visual monitoring of distribution progress, recipient counts, and animal allocation statistics.
- AI Distribution Planner: Uses AI as a tool to reason and estimate the optimal number of meat portions per animal based on total weight and local donor data.
- Firestore Centralized Registry: Unified database for managing donors, animals, and recipients with instant multi-device sync.
- Status Track & Guard: Operational workflow tracker that prevents duplicate coupon redemptions with loud visual alerts.
- Geographic Area Summaries: Hierarchical data views to monitor and prioritize qurban distribution by neighborhood blocks (RT/RW).
- Sun-Optimized Interface: Mobile-first PWA with ultra-high contrast UI designed specifically for high-glare outdoor environments.

## Style Guidelines:

- Primary Color: Deep Islamic Emerald (#15803D) providing a strong cultural anchor with high visibility in sunlight.
- Background: Clean Alabaster Green (#F0F7F1) to reduce eye strain while maintaining the core hue identity.
- Accent: Vibrant Chartreuse (#A3E635) for critical action buttons and scan confirmations, ensuring they pop against the dark green.
- Font Pairing: 'Space Grotesk' for bold, machine-like headlines and 'Inter' for highly legible body text on low-resolution devices.
- Large, chunky stroke icons (2.5px width) using Lucide-react for rapid recognition by non-technical field staff.
- Mobile-first ergonomic layout with sticky bottom navigation and 48px minimum touch targets for reliable outdoor usage.
- Functional haptic-like animations, specifically a bright fullscreen pulse when a coupon is successfully validated.
